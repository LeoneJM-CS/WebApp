Must add React and npm to personal laptop to get this to work.

<details>
  <summary>Node/npm</summary>
  
  ### Setting up Node/npm
  - Open Node.js download
  - install Node version with LTS (Long Term Support)

  ### Creating a Server
  - http.createServer()
  - ctl + C to kill server

  ### Getting Packages
  - nodemon --> "npm install nodemon --global"
  - - installs nodemon as a global package
  - underscore --> "npm install underscore"

  ### File Manipulation
  - File System -> read and write files
</details>

<details>
  <summary>React</summary>
  
  ### Setting up React
  command --> "npm create vite@latest"
  - Set framework to "React"
  - Set language to "JavaScript"
  ### Running React
  - Set to correct directory
  - npm run dev
</details>
